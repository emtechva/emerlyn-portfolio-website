# Built With Emerlyn Portfolio

Static portfolio website for Emerlyn Bequio, AI Automation Specialist.

Website brand: Built With Emerlyn  
Descriptor: AI Automation & Workflow Systems  
Tagline: Practical systems. Less manual work.

## Stack

- HTML5
- CSS3
- Vanilla JavaScript

No frameworks, packages, or build tools are required.

## Preview in VS Code

1. Open this repository in VS Code.
2. Install the Live Server extension if it is not installed yet.
3. Right-click `portfolio/index.html`.
4. Choose `Open with Live Server`.

You can also open `portfolio/index.html` directly in a browser, but Live Server is better for testing navigation, embeds, and local paths.
